# Arenito Sustentável 🌾🚜

> Projeto desenvolvido para o **Programa Agrinho Paraná 2026**
> **Categoria:** 3 - Programação (Foco em Desenvolvimento Front-End)
> **Alvo de Avaliação:** Nível 4 (Impacto Social, Complexidade, UI/UX Aplicada e Alinhamento Regional)

## 📋 Sobre o Projeto
O **Arenito Sustentável** é uma plataforma web interativa planejada especificamente para atender a realidade socioambiental e agrícola do **Noroeste do Paraná**. A região, caracterizada pela fragilidade do solo de tipo Arenito Caiuá, enfrenta sérios problemas de erosão, voçorocas e baixa retenção hídrica sob manejos incorretos em culturas fortes do local, como a mandioca, cana-de-açúcar e pastagens de pecuária.

A aplicação simula em tempo real o impacto das decisões de manejo do produtor rural, gerando um indicador analítico de sustentabilidade e recomendações práticas alinhadas ao tema de 2026 do programa: *“Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”*.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica completa visando acessibilidade.
* **CSS3:** Estilização baseada em variáveis de design corporativo/ecológico, layout fluido através de Flexbox/Grid e design 100% responsivo para smartphones e tablets do homem do campo.
* **JavaScript (ES6):** Manipulação dinâmica do DOM, tratamento de eventos sem refresh de página e algoritmos condicionais para cálculo de risco agronômico.
